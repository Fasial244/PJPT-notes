## Contents

The repository mirrors the structure of my Notion study page. Key sections:

- **Network** – basics of networking, IP addressing.
- **Kali command** – essential Kali Linux commands.
- **Bash/Python** – notes on scripting and quick examples.
- **Stages of EH** – Reconnaissance, Scanning & Enumeration, Gaining Access, Maintaining Access, Cleaning.
- **Practice** – quick reference commands and tools.
- **AD** – Active Directory notes and attack vectors.
- **Web application** – SQL and web hacking basics.
- **Wireless pentest** – wireless security topics.
- **Test day and Report** – logs, test-day notes, and report outlines.
